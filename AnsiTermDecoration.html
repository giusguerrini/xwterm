Class: AnsiTermDecoration

AnsiTermDecoration(term, canvas, params)

The AnsiTermDecoration class is an auxiliary class used by AnsiTerm to implement its default decoration. It is responsible for creating the title bar, the status bar, and the soft keyboard.

Constructor

new AnsiTermDecoration(term, canvas, params)

Parameters:
Name Type Description
term AnsiTerm

The AnsiTerm instance to decorate.
canvas HTMLCanvasElement

The canvas element to use for the terminal.
params Object

Configuration parameters for the decoration. For a full list of parameters, see ANSITERM_DEFAULTS and AnsiTerm.
